The platform supports six languages: Danish, English, Swedish, German, Arabic (RTL), and Norwegian (Bokmål).
All UI keys, AI-generated plans, changelog entries, and public pages are translated.
Arabic uses RTL layout via `document.documentElement.dir`.
Norwegian was derived from Danish with systematic Bokmål adjustments.
