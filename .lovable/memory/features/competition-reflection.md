---
name: Post-Competition Reflection
description: Offline-first post-competition reflection flow with AI action plan and 3 SMART goals; coaches see reflections in Mental tab.
type: feature
---
After each competition, athletes can complete a 4-step reflection (Result/Mood → 1–5 ratings → guided text prompts → review) accessible from the "Past Competitions" section of `/competitions` via a Reflect CTA. Results are submitted through `useOfflineCompetitionReflections` (IndexedDB cache + outbox via `competitionReflectionSyncEngine`), so reflections work offline; the AI plan is generated by the `generate-competition-reflection` edge function (Lovable AI Gateway, gemini-2.5-flash) once back online.

Stored in `competition_reflections` table with RLS allowing athletes to manage their own rows and coaches to view rows for managed or club athletes. Output schema: `summary`, `strengths[]`, `focusAreas[]`, exactly 3 `nextCompetitionGoals[]` (SMART), `mentalRoutineUpdates[]`, `recommendedAssessment`. Coach view lives in `src/components/coach/CoachAthleteReflections.tsx` and is rendered inside the Mental tab of `CoachAthleteDetail`.

Routes: `/competitions/:id/reflect` (page `src/pages/CompetitionReflection.tsx`).
