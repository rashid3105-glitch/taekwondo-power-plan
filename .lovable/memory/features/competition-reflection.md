---
name: Post-Competition Reflection
description: Offline-first post-competition reflection flow with AI action plan and 3 SMART goals; coaches see reflections in Mental tab; athletes are prompted from the Dashboard for up to 14 days after each event.
type: feature
---
After each competition, athletes complete a 4-step reflection (Result/Mood → 1–5 ratings → guided text prompts → review). It produces an AI action plan with 3 SMART goals for the next event.

**Where the athlete starts a reflection (multiple entry points):**
1. **Dashboard hub** — `<ReflectionPromptCard />` shows automatically for any past competition within the last 14 days that has no reflection yet. Dismiss persists in `sessionStorage` per competition id.
2. **Competitions page** — when something needs reflection, the "Past competitions" section moves to the **top** of the page (above the upcoming list). When all are reflected, it falls back to the bottom as a history view. Fresh (≤7 days) cards get a primary-variant pulsing CTA.
3. **Direct route** — `/competitions/:id/reflect`.

Reflections submit through `useOfflineCompetitionReflections` (IndexedDB cache + outbox via `competitionReflectionSyncEngine`). The AI plan is generated by the `generate-competition-reflection` edge function (Lovable AI Gateway, gemini-2.5-flash) once back online. Stored in `competition_reflections` (RLS: athletes manage own; coaches read for managed/club athletes). Output schema: `summary`, `strengths[]`, `focusAreas[]`, exactly 3 `nextCompetitionGoals[]` (SMART), `mentalRoutineUpdates[]`, `recommendedAssessment`.

Coach view lives in `src/components/coach/CoachAthleteReflections.tsx` and is rendered inside the Mental tab of `CoachAthleteDetail`. Coaches can also create competitions for an athlete (via `create-athlete-competition` edge function) and add private comments per reflection (`coach_reflection_comments` table).

**Components / files**
- `src/components/ReflectionPromptCard.tsx` — Dashboard nudge
- `src/components/PostCompetitionReflection.tsx` — 4-step wizard + results
- `src/components/ReflectionTrendChart.tsx` — recharts line chart of ratings over time
- `src/pages/CompetitionReflection.tsx` — route page
- `src/pages/Competitions.tsx` — past-competitions section is conditionally promoted to top
